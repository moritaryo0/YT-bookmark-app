YouTube Bookmark

お気に入りの動画を整理して、後で見たい動画をスマートに管理

YouTubeで気に入った動画や後で見たい動画をまとめておくためのWebアプリケーションです。

簡単な操作で動画をブックマークし、いつでもアクセスできるように整理できます。

動画を探す マイページ

主な機能

シンプルで使いやすい機能でYouTube動画を効率的に管理

チャンネル検索

YouTubeチャンネルIDやキーワードで動画を検索。 YouTube Data API v3を使用して最新の動画情報を取得します。

簡単ブックマーク

ワンクリックで動画をブックマークに追加。 サムネイル、タイトル、チャンネル情報を自動で保存します。

マイページ管理

保存した動画を一覧で管理。 お気に入りの動画にいつでも素早くアクセスできます。

URL直接保存

YouTube動画のURLを直接入力して保存。 検索せずに手軽にブックマークを追加できます。

技術スタック

Django学習のために制作したWebアプリケーション

このサイトはDjangoフレームワークを使ってWebサイトの基礎を構築し、 フロントエンドはClaudeが製作したモダンなデザインを採用しています。
（デザインがやけにかっこいいのはそのせいです。実際のところ,私にフロントエンドの技術はありません。私がきちんと学んだと言えるのはバックエンドのみです。）

YouTube Data API v3を使って動画情報を取得し、データベースに保存するシンプルな構造ですが、 キーワード検索やマイページでのブックマーク管理など、ユーザビリティの細かい部分にもこだわりました。
